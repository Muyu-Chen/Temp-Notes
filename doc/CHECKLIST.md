## 🏗️ 架构特点

### 1. 分层架构
```
Presentation Layer (UI)
    ↓
Business Logic Layer (业务)
    ↓
Data Access Layer (存储)
    ↓
Utility Layer (工具)
```

### 2. 模块化设计
- 7个独立的 JS 模块
- 4个独立的 CSS 模块
- 清晰的职责划分
- 低耦合高内聚

### 3. 事件驱动
- 用户操作 → 事件监听
- 事件处理 → 业务逻辑
- 数据更新 → UI 渲染
- 用户反馈 → 完成

### 4. 数据流
```
用户输入
  ↓
DOMManager (捕获)
  ↓
AppController (业务处理)
  ↓
Storage (持久化) + UIController (渲染)
  ↓
用户看到反馈
```
