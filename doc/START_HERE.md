## 📞 遇到问题？

| 问题 | 解决方案 |
|------|---------|
| 应用不加载 | 查看浏览器控制台（F12）是否有错误 |
| 数据没保存 | 检查 LocalStorage 是否被禁用 |
| 导入失败 | 确保 JSON 格式正确（从应用导出的最佳） |
| 快捷键不工作 | 部分输入法会拦截，可以用按钮代替 |
| 需要帮助 | 查看相关文档或 [INDEX.md](INDEX.md#快速查找) |
